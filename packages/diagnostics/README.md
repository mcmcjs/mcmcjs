# @mcmcjs/diagnostics

MCMC convergence diagnostics for [MCMC.js](https://github.com/mcmcjs/mcmcjs):
rank-normalized split-R-hat, bulk/tail effective sample size (ESS), Monte Carlo
standard error (MCSE), and highest-density intervals (HDI), following
Vehtari et al. (2021).

Operates on plain arrays of chains (`Float64Array[]`), so it has no dependencies.

> Early alpha — the API is not yet stable.
